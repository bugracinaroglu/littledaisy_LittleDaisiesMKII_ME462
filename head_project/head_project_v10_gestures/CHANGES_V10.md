# v10 Changes

- Added `SIGMA` and `SUNGLASSES` LCD faces.
- Added non-blocking RP2350 gesture controller.
- Added configurable `count` and `hold_ms` parameters.
- Added NOD, SUNGLASSES_NOD, SIGMA_NOD, SHAKE, LOOK_AROUND, CELEBRATE, SLEEP and WAKE_UP.
- Added face-lock priority over IMU RUNNING/DIZZY overrides.
- Reduced RUNNING face sensitivity and delayed activation.
- Added Raspberry Pi API, MANUAL keys and ROS2 gesture interfaces.
- Preserved intentional `NEUTRAL -> CURIOUS` behavior.
